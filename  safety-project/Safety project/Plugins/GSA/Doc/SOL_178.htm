Sound Transmission Between Spaces

Overview

This rule checks that walls and doors around spaces have STC rating high enough. Required STC rating depends on the privacy level of the space.

Running this rule with the default values does not give useful results; this rule has to be configured before use.

Configuring the Rule

The rule has the following parameters:

Privacy Levels

A table for privacy levels spaces with specified type and name. Privacy level is one of:

  • Inaudible (required STC rating is 55)

  • Confidential (required STC rating is 50)

  • Normal (required STC rating is 45)

  • Minimal (required STC rating is 40)

  • None

Wall STC Ratings

A table about wall types' STC ratings.'

Door STC Ratings

A table about door types' STC ratings.

Wall Offset Tolerance

A tolerance for searching walls around spaces. Walls, which are nearer than the tolerance, are taken into account.

Acceptable Gap

An acceptable gap between walls around a space.

Analyzing the Results

If a space is not totally surrounded by walls, an issue is created and the space is attached to it. If there are some surrounding walls, they are attached to the issue as information components.

If a space isn't found in the Privacy Levels table, an issue is created and the space is attached to it.

If STC ratings of surrounding  walls and doors is smaller than required, an issue is created and the space and walls and doors with too small STC rating are attached to the issue.

Reported Data

The rule does not have a report.

Rule Tools

The tools panel of the rule contains tools to help you to fill parameter tables.

Add All Space Names and Types

Adds all space name and type combinations in the model to Privacy Levels table. You have to select the privacy level for them.

Add All Wall Types

Adds all wall types in the model to Wall STC Ratings table. You have to select STC rating to them. If 'Allow Automatic STC Rating Determination' is checked, the system tries to search string 'STC' from the type name and tries to determinate the STC rating of the type.

Add All Door Types

Adds all door types in the model to Door STC Ratings table. You have to select STC rating to them. If 'Allow Automatic STC Rating Determination' is checked, the system tries to search string 'STC' from the type name and tries to determinate the STC rating of the type.
