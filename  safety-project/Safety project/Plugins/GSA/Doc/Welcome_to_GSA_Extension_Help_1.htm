Welcome to GSA Extension Help

You can familiarize yourself with Solibri Model Checker (SMC) by reading SMC Help and/or going through Getting Started tutorial.

With GSA extension for SMC you can analyze space program and other things related to spaces (various classifications, etc.). It also checks that spaces are modeled according to GSA BIM Guidelines. The analyzing process includes model creation, visualization, checking and reporting.

GSA extension includes:

  • Special properties to spaces, floors and buildings

  • Tools to calculate space usable and rentable areas

    • according to ANSI/BOMA rules (Standard Method for Measuring Floor Area in Office Buildings, ANSI/BOMA Z65.1-199)

    • or according to GSA own method

  • Tools to visualize spaces according to their ANSI/BOMA Classification

  • Creation of GSA Spatial Program BIM Report

  • GSA BIM Guide Validation rule set

  • Special rules to check and report GSA specific data:

    • Efficiency Ratios

    • Housing Plan

    • Program Reconciliation

    • Sound Transmission Between Spaces

    • Mandatory Space Data Check

    • Stacking Plan Report and Visualization

    • Walls Between Secure, Restricted and Public Spaces

    • Space Value Check

    • Area Comparison Rule

    • Space Category Consistency
