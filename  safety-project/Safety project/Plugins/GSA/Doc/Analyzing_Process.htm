Analyzing Process

To get valid results from SMC GSA extension tools, you have to prepare the BIM model for calculations. This page tells you shortly about the basic steps to do before running the analyzes.

Import IFC model

The model has to be saved as IFC before it can be read to SMC. Detailed instructions about IFC import can be found from common SMC Help, Importing an IFC file.

Take an overview of the model in 3D to gain and overall understanding about it.

DEFINE EXTERNAL WALLS

Calculation of polygon areas of spaces requires information about external walls. You can define external walls in the  Compartmentation View (Window -> Add View -> Compartmentation). Go through the external walls in every floor, and modify them manually if needed by adding and removing walls. If the model includes mezzanines in separate floors, remove external walls from those floors.

GSA BIM Guide Validation rule set includes external walls validation rule. You can also use its Tools View in External Wall Validation rule.

Check ANSI/BOMA Classification

Open ANSI/BOMA View (GSA Layout -> ANSI/BOMA) and check that ANSI/BOMA categories are correct. You can visualize spaces by occupant organizations, by locations (floors), or by ANSI/BOMA categories. You can manually change categorizing, if needed.

Check the model

Select the GSA BIM Guide Validation rule set and check the model.

Results of the first two sub rule sets, Spaces and Zones and Building Elements, gives you some general information about the model. If there are lot of errors, e.g. duplicate or overlapping spaces, the results of the rest of the analysis is likely to be incorrect.

Spatial Program Analysis sub rule checks, that the spaces has all required properties. Visualization and Reporting sub sets allows you to analyze model.

Investigate Checking Results

Investigate results of Program Analysis rule set. Most of the rules have an own report, which can be very informative.

Visualize the Model

Visualize the model from different views:

  • Polygon areas from ANSI/BOMA View

       Stacking plans from tools panels of the Stacking Plan rules

Create Report

At the end you can create GSA Spatial Program BIM Report from ANSI/BOMA View.
