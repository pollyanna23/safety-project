Space Properties

ANSI/BOMA Properties

These properties are calculated or determinate by SMC.

Basic Rentable Area

Calculation rule: Floor RU Ratio * Space Polygon Area. If the GSA Calculation method is in use, the value is the same as Usable Area.

Category

ANSI/BOMA Space Category. Same as ANSI/BOMA Space Category in GSA Space Categories. If this is empty, the value is determined by the GSA STAR Space Type according the Assignment Data Matrix No. 2 in PBS Business Assignment Guide (Office of Real Property Asset Management, May 1005). You can also set the category manually.

External Wall Status

If you the user has set external wall status to the external walls (it is done when ANSI/BOMA View is opened, or you can set it from the Compartmentation View), the value is 'Defined'. If external walls are not set, the value is 'Please, define'. That means that you should set the external walls, because usable areas cannot been reliable calculated if they are not set.

GSA BIM Area

The area calculated from the geometry of a space component by removing areas of intersecting columns and walls.

Non-assignable Area

The area is the value of Polygon Area of all Major Vertical Penetration and PBS Specific spaces.

Office Rentable Area

Calculation rule: Building RU Ratio * Space Basic Rentable Area

Polygon Area

The Polygon Area is base for the all other properties. It includes the space area and bottom area of walls around itself. The amount of wall bottom area depends on the ANSI/BOMA Space Category of the space. Ref. Area Calculation.

Rentable  Area

The area is the value of Polygon Area of all Building Common and Floor Common spaces.

Usable  Area

The area is the value of Polygon Area of all Office (and Store) spaces.

GSA Space Areas

This property set is read from IFC file. For 'full floor spaces' it should have at least property GSA Design Gross Area.

For 'normal' spaces the property set should have GSA BIM Area (used to be GSA Net Area). There can be also two optional properties GSA Usable Area and GSA Rentable Area.

GSA Space Categories

This property set is read from IFC file.

 

VAIHDA KUVA

 

 

ANSI/BOMA Space Category

An optional property. Value is a code representing one category (e.g. 01 = Office). Categories are listed in Section 5: Space Types in PBS Business Assignment Guide (Office of Real Property Asset Management, May 1005).

GSA STAR Space Category

An optional property. Value is a code representing one category (e.g. 01 = Assigned). Categories are listed in Assignment Data Matrix No. 1 in PBS Business Assignment Guide (Office of Real Property Asset Management, May 1005)

GSA STAR Space Type

A required property. Value is a code representing one STAR Type. Types are listed in Assignment Data Matrix No. 2 in PBS Business Assignment Guide (Office of Real Property Asset Management, May 1005)

Space Occupant Properties

This property set is read from IFC file.

Value for Occupant Organization Name is required. Other properties are optional.

Space Zones

This property set is read from IFC file.

All properties are optional.
