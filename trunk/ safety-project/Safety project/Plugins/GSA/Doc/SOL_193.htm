Mandatory Space Data Check

Overview

This rule checks that all spaces have mandatory space data. Obligatory data from in External Space Data is Agency Bureau Abbreviation, Organization Unit Code, and STAR Space Type. The rule crates issues of missing data. It also checks, that every space has an ANSI/BOMA Category and creates an issue for each space wihtout it. It also creates issus from ANSI/BOMA Categories, which shouldn't be used.

Configuring the Rule

The rule doesn't have any parameters.

Analyzing the Results

All spaces, which don't have mandatory space data or ANSI/BOM Category is UNDEFINED creates an issue. Spaces with same  missing data are collected to same category.

If Store Area or Floor Common categories are used, an issue is created. They shouldn't be used according to PBS Business Assignment Guide.

If there are Vertical Penetration Spaces, whose net area is less than 9 sq ft, an issue is created. The limit is set in  PBS Business Assignment Guide.

Reported Data

The rule does not have a report.

Rule Tools

The rule doesn't have a tools panel.
