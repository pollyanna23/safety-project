Stacking Plan Report and Visualization

Overview

This rule creates report about usable and rentable areas and allows you to visualize spaces by different properties.

Configuring the Rule

The rule has the following parameters:

Visualization Classification

You can select what property is used for categorizing spaces. The possible properties are:

  • Space Name

  • Space Number

  • ANSI/BOMA Space Category

  • Occupant Organization Code

  • Occupant Organization Name

  • Occupant Organization Abbreviation

  • Occupant Sub-Organization Code

  • Occupant Billing ID

  • Security Zone

  • Privacy Zone

  • Preservation Zone

  • GSA STAR Space Type

  • GSA STAR Space Category

  • Space Group Name (name of the space group the space is included in)

Analyzing the Results

The rule doesn't check anything.

Reported Data

The rule creates a report containing Office Polygon Area, Store Polygon Area, Rentable - Building Common Area, Rentable - Floor Common Area, and Office rentable Area for all spaces with the same classifier floor by floor.

Rule Tools

The rule tools panel has two different hierarchies, Location Hierarchy and Space Classifier Hierarchy. When you select a floor, a space classifier or an individual space from the three, spaces are visualized in the 3D View with the color specified in the table. You can change the color by double clicking the color cell. You can also set spaces in a hierarchy shown or not shown from the Is Shown cell.
